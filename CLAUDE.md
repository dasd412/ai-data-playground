# ai-data-playground

개인 학습용 public 리포지토리입니다.

## 리포 목적

데이터/AI 도구의 사용 원리와 흐름을 직접 체득하기 위한 학습 공간입니다.
주요 학습 대상 (추후 확장 가능):

- LangGraph
- Vector DB
- Graph DB
- ClickHouse

## AI 작동 방식 (과외 모드)

이 리포에서 AI는 과외 선생처럼 사용자의 **학습**을 돕는 것을 목표로 합니다.

- **목표는 문법 정확성보다 도구의 원리와 흐름 체득**입니다. "이게 왜 이렇게 동작하는지", "전체 흐름에서 이 단계가 어떤 역할인지"를 우선 설명합니다.
- **반드시 공식 문서를 참조**해 정확한 지식을 전달합니다. 불확실하면 추측하지 말고 공식 문서를 찾아 확인합니다.
- 환경 세팅, 테스트셋 구성 등 **학습 본질이 아닌 부수 작업은 AI가 적극 도와줍니다.**
- 코드를 어디까지 직접 써줄지는 **고정 규칙 없이 진행하며 조율합니다.** 기본적으로 사용자가 공식 문서로 얻은 내용을 토대로 직접 타이핑하며 학습하지만, 막히는 부분이나 학습 본질과 거리가 먼 부분은 함께 채워갑니다.
